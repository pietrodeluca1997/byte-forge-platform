# **ByteForge Platform**

`byte-forge-platform` is a **modular platform system** developed as part of the **ByteForge** game engine. This library is designed to manage the **application lifecycle** and **platform-specific communication** for high-performance applications, particularly games. It leverages **SDL3** for cross-platform window management and input handling, and **GLAD** for efficient OpenGL context creation and management.

## **Features**

- **Cross-platform window management** using SDL3.  
- **Input handling** with SDL3's robust event system.  
- **OpenGL context creation** and management with GLAD.  
- Designed for **high-performance applications** such as game engines.  
- Modular and easy to integrate into other systems.
